🖥️ Digital Service Center Website

A professional single-page website built for Cyber Cafes and Digital Service Centers.
Customers can browse 18+ government & utility services, submit requests via WhatsApp,
and owners earn through Google AdSense ads.

✨ Features
- 25+ services across Identity, Government, Online Work & Other categories
- Live search & category filter system
- 3-step service request flow (Details → Documents → WhatsApp Send)
- Auto-generated WhatsApp message with customer details
- Google AdSense ads (inline + sticky sidebar)
- Fully responsive — mobile, tablet & desktop
- Smooth animations & modern gradient UI

🛠️ Built With
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Syne + DM Sans)
- Google AdSense

🚀 Setup
1. Clone the repo
2. Replace ca-pub-XXXXXXXX with your AdSense Publisher ID
3. Replace ad slot IDs with your real AdSense slot values
4. Update WhatsApp number (917384899003) in the script
5. Upload to your hosting / GitHub Pages
